# خريطة طريق ساك — الأعمال المؤجلة

> أنشئت: 21 يونيو 2026  
> هذا الملف يوثق كل الأعمال المستقبلية بعد اكتمال صفحة الهبوط (Landing Page).  
> أولوية العمل: **صفحة الهبوط** ← ثم **بوابات الدخول** ← ثم **لوحات التحكم**.

---

## WS9 — بوابات الدخول (Auth Pages)
**الوصف:** صفحات تسجيل الدخول وإنشاء الحساب للنظام الأساسي الأربعة.

| المسار | الحالة | ملاحظات |
|--------|--------|---------|
| `/user/login` | تم عمل Stub | يحتاج صفحة كاملة + RTL + Arabic |
| `/user/register` | TODO | تسجيل الأفراد |
| `/user/forgot-password` | TODO | استعادة كلمة المرور |
| `/agent/login` | تم عمل Stub | لوكلاء الخدمة |
| `/agent/register` | TODO | تسجيل الوكلاء |
| `/merchant/login` | تم عمل Stub | للتجار |
| `/merchant/register` | TODO | تسجيل التجار |
| `/company/login` | تم عمل Stub | للشركات |
| `/company/register` | TODO | تسجيل الشركات |

**المتطلبات:**
- RTL عربي بالكامل
- تحقق ثنائي (2FA)
- تسجيل عبر البريد/الهاتف
- OAuth (Google/Apple) — لاحقاً
- واجهة موحدة (Glassmorphism) تتناسق مع Landing Page

---

## WS11 — واجهة لوحة التحكم الرئيسية (Dashboard Shell)
**الوصف:** الهيكل الأساسي للوحة التحكم بعد تسجيل الدخول — شريط جانبي، رأس، محتوى.

**المكونات:**
- `(dashboard)/layout.tsx` — هيكل رئيسي مع Sidebar + Header
- `(dashboard)/page.tsx` — نظرة عامة مع Widgets
- Sidebar: أيقونات + روابط + صورة مستخدم + قائمة تنقل
- Header: إشعارات، بحث، قائمة مستخدم
- 4 نُسخ من لوحة التحكم (User, Agent, Merchant, Company)

---

## WS12 — لوحة تحكم المستخدم (User Dashboard)
- الرصيد والمعاملات الأخيرة
- بطاقات الدفع (رقمية/بلاستيكية)
- تحويلات سريعة (بين حسابات ساك)
- نقاط الولاء
- كشف الحساب
- إعدادات الحساب + الملف الشخصي

---

## WS13 — لوحة تحكم الوكيل (Agent Dashboard)
- إدارة العملاء
- طلبات تفعيل البطاقات
- سجل العمولات
- تقارير الأداء
- طلبات الدعم

---

## WS14 — لوحة تحكم التاجر (Merchant Dashboard)
- نظرة عامة على المبيعات
- إدارة المدفوعات (QR كود، روابط دفع، نقاط بيع)
- تسوية الحسابات
- إدارة المنتجات/الفواتير
- تقارير المبيعات

---

## WS15 — لوحة تحكم الشركة (Company Dashboard)
- إدارة الموظفين والبطاقات
- صرف الرواتب
- تقارير مالية متقدمة
- إدارة الصلاحيات
- تكامل API

---

## أعمال إضافية مؤجلة

| العنصر | الأولوية | ملاحظات |
|--------|---------|---------|
| Backend API Integration | عالية | ربط كل الواجهات بالـ Backend |
| Mobile App (Flutter/RN) | متوسطة | تطبيق هاتف موازٍ |
| OTP/SMS Service | عالية | خدمة رسائل للتحقق |
| Email Service | متوسطة | إشعارات البريد الإلكتروني |
| Admin Panel | متوسطة | لوحة تحكم المشرف العام |
| Internationalization (i18n) | منخفضة | دعم EN/TR/KU |
| PWA Support | منخفضة | تحويل الموقع لتطبيق ويب |
| Dark/Light Theme Toggle | منخفضة | تبديل الوضع (حالياً Dark فقط) |

---

> **ملاحظة:** جميع التصاميم تتبع نمط Glassmorphism + Dark Mode (#0A0A0A) + RTL كما هو مطبق في صفحة الهبوط.
> المكتبات الأساسية: Next.js 16 + TypeScript + Tailwind v4 + Zustand + GSAP + Three.js.
