# Design System — FreelanceHub
> Visual identity, color palette, typography, spacing, and brand elements.

## Brand Concept
- **Name meaning:** FreelanceHub — مركز العمل الحر
- **Logo concept:** Circle with star (hub) + pen tip, clean blue gradient
- **Brand personality:** Professional, Clean, Empowering, Modern, Simple

## Color Palette
- **Primary:** `#3B82F6` — Blue 500, productivity + trust
- **Secondary:** `#0D9488` — Teal 600, growth + stability
- **Accent:** `#6366F1` — Indigo 500, creativity
- **Neutral:** `#1F2937` — Gray 800 (text), `#F9FAFB` (bg)
- **Semantic:** Success `#10B981` · Warning `#F59E0B` · Error `#EF4444`

## Typography
- **Headings:** Inter — sizes: 24/20/18/16px
- **Body:** Inter — 14px
- **Arabic:** Noto Kufi Arabic — invoices, proposals, app UI

## Spacing
- Base unit: 4px/8px
- Padding: 16/24/32px
- Border radius: 8px

## Iconography
- Style: Outline
- Library: Lucide
