# JOURNEY MAP — EduCloud LMS (SAAS-006)
> Owner: Journey Architect · Gate 1 · Persona: سامي — مدرب مهارات

## المسار (Mermaid)
```mermaid
flowchart LR
  trigger[مدرب يريد منصة تعليمية] --> signup[تسجيل في EduCloud]
  signup --> create_course[إنشاء دورة جديدة]
  create_course --> add_content[إضافة محتوى: فيديو-PDF-نص]
  add_content --> add_quiz[إضافة اختبارات]
  add_quiz --> publish[نشر الدورة]
  publish --> enroll[تسجيل طلاب]
  enroll --> learn[الطلاب يتابعون الدروس]
  learn --> take_quiz[أداء الاختبارات]
  take_quiz --> grade[تصحيح وتقييم]
  grade --> certificate[إصدار شهادة]
  certificate --> goal[إتمام الدورة بنجاح]
  
  add_content -.-> video_error[فشل رفع الفيديو] --> retry_upload[إعادة المحاولة]
  take_quiz -.-> fail[رسوب في الاختبار] --> retake[إعادة المحاولة]
  enroll -.-> payment_fail[فشل الدفع] --> retry_payment[إعادة الدفع]
```

## سجل الاحتكاك المرتب
1. [High] رفع الفيديو بطيء → حل: رفع تدريجي مع شريط تقدم + تشفير تلقائي (Screen 2)
2. [High] لا اختبارات آلية التصحيح → حل: بنك أسئلة مع تصحيح تلقائي (Screen 3)
3. [Med] صعوبة تتبع تقدم الطلاب → حل: لوحة تقدم لكل طالب (Screen 5)
4. [Med] الشهادات اليدوية غير احترافية → حل: شهادات PDF مع QR (Screen 6)
5. [Low] لا تقارير أداء للدورات → حل: تقارير إتمام ونسب نجاح (Screen 7)
