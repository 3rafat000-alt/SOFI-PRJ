# Design System — SchoolEase
> Visual identity, color palette, typography, spacing, and brand elements.

## Brand Concept
- **Name meaning:** SchoolEase — المدرسة السهلة
- **Logo concept:** Open book + graduation cap in purple, academic shield outline
- **Brand personality:** Academic, Trustworthy, Modern, Friendly, Inspiring

## Color Palette
- **Primary:** `#7C3AED` — Violet 600, wisdom + academia
- **Secondary:** `#4F46E5` — Indigo 600, depth + focus
- **Accent:** `#A855F7` — Purple 500, creativity + inspiration
- **Neutral:** `#1E293B` — Slate 800 (text), `#F5F3FF` (bg)
- **Semantic:** Success `#10B981` · Warning `#F59E0B` · Error `#EF4444`

## Typography
- **Headings:** Inter — sizes: 24/20/18/16px
- **Body:** Inter — 14px
- **Arabic:** Noto Kufi Arabic — report cards, parent communication, grade descriptions

## Spacing
- Base unit: 4px/8px
- Padding: 16/24/32px
- Border radius: 8px

## Iconography
- Style: Outline
- Library: Lucide
