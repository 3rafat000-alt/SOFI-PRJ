# Design System — WeddingOrg
> Visual identity, color palette, typography, spacing, and brand elements.

## Brand Concept
- **Name meaning:** WeddingOrg — تنظيم الزفاف
- **Logo concept:** Two rings + heart in rose gold, elegant serif script
- **Brand personality:** Romantic, Elegant, Luxurious, Joyful, Dreamy

## Color Palette
- **Primary:** `#E11D48` — Rose 600, romance + passion
- **Secondary:** `#D97706` — Amber/Gold 600, luxury + celebration
- **Accent:** `#BE185D` — Pink 700, elegance + depth
- **Neutral:** `#1F2937` — Gray 800 (text), `#FFF1F2` (bg)
- **Semantic:** Success `#10B981` · Warning `#F59E0B` · Error `#EF4444`

## Typography
- **Headings:** Playfair Display — sizes: 28/24/20/18px (elegant serif)
- **Body:** Inter — 14px
- **Arabic:** Noto Kufi Arabic — guest names, invitations, budget items

## Spacing
- Base unit: 4px/8px
- Padding: 16/24/32px
- Border radius: 12px (softer for wedding theme)

## Iconography
- Style: Outline, with heart/star accents
- Library: Lucide + custom wedding icons (rings, cake, flowers)
