# JOURNEY MAP — ClinicFlow (SAAS-002)
> Owner: Journey Architect · Gate 1 · Persona: د. ليلى — طبيبة أسنان

## المسار (Mermaid)
```mermaid
flowchart LR
  trigger[مريض يتصل لحجز موعد] --> register[تسجيل المريض جديد؟]
  register -->|نعم| new_patient[إضافة ملف مريض]
  register -->|لا| find_patient[بحث عن مريض]
  new_patient --> book_appointment[حجز موعد]
  find_patient --> book_appointment
  book_appointment --> confirm[تأكيد الحجز]
  confirm --> reminder[تذكير تلقائي قبل 24 ساعة]
  reminder --> checkin[حضور المريض]
  checkin --> consult[الكشف الطبي]
  consult --> diagnosis[تسجيل التشخيص]
  diagnosis --> prescription[وصف علاج]
  prescription --> invoice[إصدار فاتورة]
  invoice --> goal[إتمام الزيارة]
  
  confirm -.-> cancel[إلغاء الموعد] --> reschedule[إعادة جدولة]
  checkin -.-> late[تأخر المريض] --> push_slot[تعديل الجدول]
  diagnosis -.-> referral[تحويل لأخصائي] --> followup[متابعة لاحقة]
```

## سجل الاحتكاك المرتب
1. [High] تخلف المرضى عن المواعيد (no-show 30%) → حل: تذكير ذكي قبل 24 ساعة وساعة (Screen 1)
2. [High] فقدان السجلات الورقية → حل: EMR سحابي مع بحث فوري (Screen 3)
3. [Med] صعوبة إيجاد ملفات المرضى بسرعة → حل: بحث باسم أو هاتف أو رقم ملف (Screen 2)
4. [Med] ازدواجية الحجز على نفس الوقت → حل: التحقق الآلي من تعارض المواعيد (Screen 1)
5. [Low] إصدار الفواتير اليدوي → حل: فواتير إلكترونية تلقائية (Screen 4)
