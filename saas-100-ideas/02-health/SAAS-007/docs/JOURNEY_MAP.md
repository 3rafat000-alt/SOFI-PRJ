# JOURNEY MAP — FitZone Pro (SAAS-007)
> Owner: Journey Architect · Gate 1 · Persona: فيصل — صاحب نادي رياضي

## المسار (Mermaid)
```mermaid
flowchart LR
  trigger[شخص يريد الانضمام للنادي] --> browse[تصفح الخطط]
  browse --> signup[تسجيل العضوية]
  signup --> payment[دفع الاشتراك]
  payment --> activate[تفعيل العضوية]
  activate --> visit[أول زيارة]
  visit --> class[حجز حصة]
  class --> attend[حضور الحصة]
  attend --> track[تسجيل التمرين]
  track --> progress[متابعة التقدم]
  progress --> renew[تجديد العضوية]
  renew --> goal[عضو دائم]
  
  payment -.-> decline[رفض الدفع] --> retry_payment[إعادة المحاولة]
  class -.-> full[الحصة ممتلئة] --> waitlist[قائمة انتظار]
  attend -.-> no_show[غياب بدون إلغاء] --> penalty[تطبيق سياسة الغياب]
  renew -.-> churn[عدم التجديد] --> offer[عرض ترويجي]
```

## سجل الاحتكاك المرتب
1. [High] الحصص تمتلئ بسرعة → حل: قائمة انتظار + إشعار بفتح مكان (Screen 3)
2. [High] انخفاض التجديد في الصيف → حل: برامج ولاء + خصومات سنوية (Screen 6)
3. [Med] لا توجد طريقة لمعرفة أوقات الهدوء → حل: رسم بياني للازدحام (Screen 7)
4. [Med] صعوبة تسجيل التمارين → حل: قوالب تمارين سريعة + مؤقت (Screen 5)
5. [Low] نسيان إحضار بطاقة العضوية → حل: بطاقة رقمية في التطبيق (Screen 4)
