# JOURNEY MAP — LaundryHub (SAAS-015)
> Owner: Journey Architect · Gate 1 · Persona: محمد الدوسري

## Flow (Mermaid)
```mermaid
flowchart LR
  A[تسجيل الدخول] --> B[لوحة الطلبات]
  B --> C{طلب جديد}
  C --> D[تسجيل بيانات العميل]
  D --> E[فرز القطع وتسعيرها]
  E --> F[طباعة باركود]
  F --> G[مرحلة الغسيل/التنظيف]
  G --> H[مرحلة الكي والتجهيز]
  H --> I[التغليف والفرز حسب التوصيل]
  I --> J[توصيل الطلب]
  J --> K[إتمام الطلب]
  G -.-> L1{قطعة تالفة}
  L1 --> L2[إشعار العميل والتعويض]
  L2 --> G
  J -.-> M1{تأخير توصيل}
  M1 --> M2[إشعار العميل بالتأخير]
  M2 --> J
  D -.-> D1{خطأ: رقم العميل غير موجود}
  D1 --> D2[تسجيل عميل جديد]
  D2 --> D
```

## Ranked Friction Log
1. [High] ضياع القطع أو اختلاط طلبات العملاء
2. [High] عدم وضوح حالة الطلب للعميل (أين قطعة الملابس حالياً؟)
3. [Med] صعوبة جدولة توصيل متعددة في نفس الحي
4. [Med] بطء إدخال قطع الملابس الكثيرة يدوياً
5. [Low] عدم وجود قوائم أسعار مبرمجة مسبقاً حسب نوع القطعة

**Rule:** Every later feature MUST trace to a stage above.
