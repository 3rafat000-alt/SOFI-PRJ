# WCAG 2.2 AA Matrix — AlertHub
| Criteria | Target | Status | Notes |
|----------|--------|--------|-------|
| Contrast ≥ 4.5:1 | Pass | ✅ | Severity badges use high-contrast red/amber/blue |
| Focus visible | All interactive | ✅ | Custom focus ring (3px offset) |
| Screen-reader labels | All inputs | ✅ | aria-label on icon buttons, status announcements |
| Keyboard navigation | Full app | ✅ | Tab through alerts, Enter to expand, Esc to close |
| Error identification | Forms | ✅ | Inline error messages with icon + text |
| Status messages | Live regions | ✅ | Alert count updates announced via aria-live |
| Touch target ≥ 44px | Mobile | ✅ | Action buttons min 48px |
| Motion reduction | Animations | ✅ | prefers-reduced-motion disables pulse |
| Color not sole meaning | Badges | ✅ | Icon + text + color for severity |
| Heading hierarchy | All pages | ✅ | h1 → h2 → h3 single skip |
