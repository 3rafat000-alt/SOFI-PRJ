# WCAG 2.2 AA Matrix — ChatFlow
| Criteria | Target | Status | Notes |
|----------|--------|--------|-------|
| Contrast ≥ 4.5:1 | Pass | ✅ | Primary #2563EB on white = 6.5:1 |
| Focus visible | All interactive | ✅ | 3px ring #2563EB |
| Screen-reader labels | Widget | ✅ | aria-label on bubble, messages announced |
| Keyboard nav | Agent workspace | ✅ | Tab through conversations, Enter to select |
| Touch target ≥ 44px | Quick replies | ✅ | Chips min 44px height |
| Live region updates | New messages | ✅ | aria-live="polite" for new messages |
| Color not sole meaning | Status badges | ✅ | Text + icon + color |
| Focus trap | Chat widget | ✅ | Trap when open, Esc to close |
| Error identification | Forms | ✅ | Inline validation |
| Timing adjustable | Offline form | ✅ | No auto-logout during typing |
